
# 🌌 Nexus Admin - Product Architect Dashboard

A high-performance, futuristic **Inventory Management Dashboard** built with a "Glassmorphism" aesthetic. Developed by **[abuki-dev](https://github.com/abuki-dev)**, this application allows architects to configure, deploy, and manage digital assets with a real-time holographic monitor.

## ✨ Features

### 📡 Live Monitor System
Experience a real-time "Data Stream" preview. As you input the product name, price, and description, the **Holographic Preview** on the right updates instantly, showing exactly how the asset will appear before deployment.

### 💎 Premium UI/UX
- **Glassmorphism:** Deep navy radial backgrounds with `backdrop-filter` blur effects
- **Micro-interactions:** Smooth CSS transitions, animated upload icons, and hover-glow effects on inventory cards
- **Responsive Core:** Optimized for all screen sizes using a custom 12-column Tailwind grid

### 🛠 Technical Specs
- **State Management:** Uses `Vanilla JS` and `LocalStorage` for persistent data storage
- **Image Processing:** Real-time `FileReader` stream for instant visual previews
- **CRUD Operations:** Full Create, Read, Update, and Delete capabilities
